\section{\mysidestyle Professional\\Experience}

\textbf{CERN}, Geneva, Switzerland \hfill \textbf{November 2013 -- Present}\\
\textsl{Ph.D. Student, ATLAS Experiment}
\begin{list2}
  \item Built up an analysis framework from scratch to search for strong production of supersymmetric particles by identifying boosted stop squarks. This framework is also used by two other analysis groups performing electroweak searches for higgsinos. This analysis is currently on-going for 2017 data.
  \item Investigating jet-area based pile-up suppression techniques applied to jets in the forward region of the ATLAS detector in high pileup environments at HL-LHC
  \item Prototyping a convolutional neural network using the ATLAS calorimeter data as a 2D image to better understand pileup at the trigger level
  \item Performing physics studies for gFEX, the instrumentation and hardware upgrades that I am involved in. These studies include identifying subjets from trigger tower information, pileup mitigation techniques, and parameterizing trigger efficiency and rates for the online trigger objects I defined.
  \item Working on analysis preservation efforts within ATLAS to ensure reproducibility of results and allow for reinterpretation by theorists
  \item Spearheading the effort for embedded processor design within the ATLAS experiment
  \item Created and currently maintain an analysis framework for general physics analyses within ATLAS including Standard Model searches, SUSY, Exotics, Higgs, and Trigger-Level analysis: \href{https://xaodanahelpers.readthedocs.io/en/master/}{xAODAnaHelpers}
  \item Built a python tool to scan the phase-space of an analysis to identify performant variables to discriminate signal over background: \href{https://github.com/kratsg/Optimization}{root-optimize}
  \item Development of a python framework that combines ROOT and NumPy: \href{http://scikit-hep.org/root_numpy/}{root\_numpy}
  \item Worked on electronic instrumentation to improve the ATLAS Trigger system in Run 3 and beyond for boosted objects: \href{https://gfex.cern.ch/}{gFEX: global feature extraction}
  \item Editor of the gFEX Final Design Report describing the technical requirements and needs of gFEX in the ATLAS Calorimeter ecosystem
  \item Developed an OpenEmbedded firmware layer for compiling a linux kernel from scratch to be installed on instrumentation in the ATLAS experiment: \href{https://github.com/kratsg/meta-l1calo}{meta-l1calo}
  \item Designed and implemented a software layer for slow-control and monitoring of embedded processors: \href{https://github.com/kratsg/ironman}{ironman}
  \item Contributed software to ATLAS for applying reclustering techniques at the analysis-level, making reclustering an easy and popular technique to use
\end{list2}
